# Projeto Chat com Validação HMAC
O conceito implementado neste projeto é similar a um IRC (Internet Relay Chat), ou seja, trata-se de um sistema de bate-papo baseado em texto, porém, com uma funcionalidade extra de validação de mensagens através de criptografia (hashing).
